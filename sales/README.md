# Future Interns - Machine Learning Internship
## Task 1: Sales & Demand Forecasting

### 🚀 Project Overview
This repository contains the implementation for **Task 1: Sales & Demand Forecasting** as part of the Future Interns Machine Learning Internship.

**Goal:** Predict future sales and demand using historical data to aid business decision-making.

### 🛠️ Key Skills & Tools
- Time-series analysis
- Regression Models
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)

### 📂 Project Structure
- `sales_forecasting.ipynb`: Main Jupyter Notebook with data simulation, EDA, and modeling.
- `requirements.txt`: Python dependencies required to run the project.

### ⚙️ Setup & Installation

1. **Clone the repository** (if applicable) or navigate to the project folder.

2. **Install Dependencies**:
   Ensure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Notebook**:
   Launch Jupyter Notebook to view and execute the analysis:
   ```bash
   jupyter notebook sales_forecasting.ipynb
   ```

### 📊 Methodology
1. **Data Generation**: Creating a synthetic dataset to simulate daily sales data.
2. **Exploratory Data Analysis (EDA)**: Visualizing trends, seasonality, and patterns.
3. **Model Building**: Implementing a regression model (e.g., Linear Regression or Random Forest) to forecast demand.
4. **Evaluation**: Assessing model performance using metrics like MAE and RMSE.
