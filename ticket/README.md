# Task 2: Support Ticket Classification

## 📌 Project Overview
This project involves building a **Machine Learning model** to automatically classify customer support tickets into different categories (e.g., *Billing, Technical Support, Refund, General Question*). This helps in routing tickets to the right department, improving response times.

## 🛠️ Key Skills & Tools
- **Python**: Core programming language.
- **NLP (Natural Language Processing)**: Text preprocessing, tokenization, stopword removal.
- **Scikit-learn**: Model building (Naive Bayes/SVM) and evaluation.
- **Data Visualization**: Matplotlib and Seaborn for insights.

## 📂 Project Structure
- `ticket_classification.ipynb`: The main notebook containing the code.
- `project_report.md`: A summary report of the project.
- `requirements.txt`: List of dependencies.

## 🚀 How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Open the notebook:
   ```bash
   jupyter notebook ticket_classification.ipynb
   ```
3. Run all cells to see the text classification pipeline in action.
